## What I'm Working On

**AI Voice Companion Robot System**  
Building the full software pipeline for an AI voice companion robot, covering both the **cloud backend** and the **ESP32 device side**. The system integrates **wake word detection**, **ASR**, **TTS**, **LLM interaction**, **RAG**, **voiceprint recognition**, **camera capture**, and **servo control**, with real-time communication over **WebSocket** and **MQTT**.

**RoboMaster Vision & Robotics**  
Led the vision team and served as project lead for RoboMaster robotics development. Built and optimized an auto-aim and target tracking system using **C++**, **OpenCV**, **YOLOv5**, **ROS**, and **CUDA**, while also working on sensor fusion, industrial camera calibration, and system integration with motion control.

**Intelligent Inventory & Management System**  
Built a full-stack warehouse inventory system based on **machine vision** and **radar modeling**, combining **YOLOv5**, **ROS**, **Flask**, **Vue**, and **MySQL** for automated item recognition, counting, and visualization.

**Garbage Recognition Platform**  
Developed a scalable garbage recognition system based on **YOLOv5 + CLIP + Milvus**, transforming traditional fixed-category detection into an **embedding + vector search** workflow for more flexible and extensible classification.
